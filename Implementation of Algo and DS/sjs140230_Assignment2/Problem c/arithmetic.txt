To run on windows machine:

1. Open cmd
2. Compile using java compiler as javac arithmetic.java
3. After compiling, run the program using command java arithmetic
4. The program will ask two numbers. Enter the numbers
5. Next it will ask for the operator. Enter the operator
6. The output will be displayed.

To run on eclipse IDE

1. Open the program in eclipse.
2. Compile and run the program.
3. The program will ask two numbers. Enter the numbers
4. Next it will ask for the operator. Enter the operator
5. The output will be displayed.
